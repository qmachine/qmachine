HPC + WWW = QM


Mix High Performance Computing with the World Wide Web and you'll get QMachine, a web service that can use ordinary browsers to execute distributed workflows — without installing anything.

Learn more »

Submitter Example

Open your browser's developer console and enter the following:

QM.puts(QM.submit(2, "(x) -> x + 2"));

You may need to volunteer to run it, though!
